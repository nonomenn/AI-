## 1. ブランドの核

- **ブランド名**：[　未定　]
- **一言でいうと**：流行を追う音楽ではなく、**「5年後も聴かれる曲」**を作るレーベル。
- **約束**：聴いた人の「特定の記憶や情景」にそっと寄り添う。派手さより余韻。
- **判断の最上位基準**：この曲は5年後もプレイリストに残るか？ 感情・ストーリー・歌詞・メロディ・世界観のどれかが、時間が経っても色褪せないか？

## 2. 目指す感情体験

- リスナーが感じてほしいもの：**共感・切なさ・あたたかさ・ふとした情景の想起**。
- 避けたいもの：使い捨て感、意味の薄い盛り上げ、記号的なだけの恋愛描写。
- 「もう一度聴きたい」を、刺激ではなく**余韻**で作る。

## 3. リスナー像

- 中心：10代後半〜20代。通学・通勤、夜のひとり時間、帰り道に音楽を聴く人。
- シーン：イヤホンで一人で聴く瞬間。情景が浮かぶ曲を求めている。

## 4. ジャンル方針

| ジャンル | 比率 | 立ち位置 |
|---|---|---|
| ラブソング | 約70% | 主軸。片思い・遠距離・再会・失恋など具体的な状況を描く |
| 応援ソング | 約20% | 押しつけない応援。挫折や迷いに寄り添う視点 |
| ボカロ系 | 約10% | 世界観・言葉の実験ができる枠。ただしブランドトーンは保つ |

- EDM / HipHop / Trap / K-POP / 演歌 / ジャズ 等は**制作対象外**。提案もしない。

## 5. 歌詞のトーン（Lyrics Agentの指針）

- **具体 > 抽象**：「好き」を連呼せず、情景（時間・場所・小道具）で感情を立ち上げる。
- **日本語の自然さ最優先**：AIっぽい説明調・機械的な比喩を避ける。
- 主人公の心情が曲の中で**変化**する（同じ地点で終わらない）。
- サビに、耳に残る一行を必ず置く。
- 禁止：抽象だけの歌詞 / 意味のない英語 / 語尾の3連続 / テンプレ恋愛表現のみ / 既存曲の酷似。

## 6. サウンドの方向性（Composition / Suno Prompt Agentの指針）

- 主軸楽器：ピアノ、アコースティックギター、ストリングス。生楽器の温度感。
- ボーカル：等身大で親密な歌声。作り込みすぎない。
- テンポ：ミディアム〜スロー中心（バラード〜ミディアムポップ）。
- 展開：ラスサビで一段景色が変わる設計（転調・ストリングス追加・オクターブ上げ等）。
- 音圧を上げすぎて余韻を潰さない。

## 7. ビジュアル / 発信のトーン（Branding / Release Agentの指針）

- ジャケット：色数を抑えた情景系。人物より「風景・光・余白」。
- 配信/SNS文言：煽らない。曲の背景ストーリーを短く添える。
- 誇大表現・虚偽の実績を書かない。
- 生成物に実在人物・既存キャラクター・著作物を模倣させない。

## 8. コンプライアンス方針

- AI支援であることを配信時に正直に開示する（DDEX等のAI開示に対応）。
- 配信は間隔を空け、重複・水増しをしない（スパム検知回避＝資産保全）。
- 既存楽曲の歌詞・メロディを模倣しない。オリジナルのみ。

## 9. 更新ルール

- このガイドラインを更新したら日付とバージョンを上げ、変更点を下記に記録する。
- Learning Agentが月次で改善提案を出した場合、ここに反映するか運営者が判断する。
