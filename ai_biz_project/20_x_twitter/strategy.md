# AI×事業効率化 X（Twitter）戦略

## 役割
Instagram（感情・信頼構築）とは別軸で、X専用コンテンツで専門性と認知を広げる。
テキスト × 図解の組み合わせで「AI×ビジネス」領域での影響力を作る。

## 主力プラットフォームとの違い

| | Instagram | X（Twitter）|
|---|---|---|
| 目的 | 信頼構築 → DM → 成約 | 認知拡大 → 専門性訴求 → フォロワー獲得 |
| 強み | 世界観・ビジュアル | 拡散・即時性・議論 |
| コンテンツ | Reels・フィード投稿 | ツイート・図解・スレッド |
| ターゲット | 経営者・個人事業主 | 経営者 + ビジネスパーソン全般 |

## コンテンツ設計

### 投稿タイプ（ローテーション）

1. **図解ツイート**（週2〜3本）
   - 「〇〇業のAI活用マップ」
   - 「ChatGPT導入の3ステップ」
   - 画像1枚 + 解説テキスト

2. **知識・ノウハウツイート**（週3〜4本）
   - 「AI導入でよくある失敗5選」
   - 「中小企業がAIを使うべき業務ランキング」
   - 単体ツイート or スレッド形式

3. **事例・実績ツイート**（週1〜2本）
   - 「〇〇業のクライアントがAI導入後△△が改善した」
   - 具体的な数字を入れると拡散しやすい

4. **意見・考察ツイート**（週1〜2本）
   - AIトレンドへの見解
   - 「〇〇について正直に言う」系

### 投稿頻度
- 1日2〜3投稿（平日中心）
- 図解は週2〜3枚ペースで制作

### ベスト投稿時間（日本）
- 7:00〜8:00（通勤時間）
- 12:00〜13:00（昼休み）
- 21:00〜22:00（夜）

## ファネル設計

```
Xツイート（認知・専門性）
    ↓
プロフィールへ誘導
    ↓
固定ツイート（Instagram or 相談申し込みへ）
    ↓
DM or Instagram フォロー
    ↓
成約
```

## KPI目標（30日）

| 指標 | 目標 | 見直し基準 |
|---|---|---|
| フォロワー | 500名+ | 200名未満 |
| インプレッション/投稿 | 1,000+ | 300未満 |
| エンゲージメント率 | 3%+ | 1%未満 |
| プロフィールクリック | 月200+ | 月50未満 |

## 図解制作ルール

- サイズ：1080×1080px（正方形）or 1200×675px（横長）
- フォント：視認性重視（Noto Sans JP 推奨）
- カラー：ブランドカラーに統一（`.claude/skills/brand-voice/` 参照）
- 1枚に情報を詰め込みすぎない（3〜5ポイントまで）

## アカウント情報
- X: （開設時に追記）
- 固定ツイート：Instagram誘導 or 相談申し込みリンク

## ブランドボイス（X専用）
- Instagramより少しカジュアルでOK
- 専門用語は使うが、必ず噛み砕いて説明
- 断定的な表現で読みやすく（「〜です」より「〜する」）
