# AI事業加速 マスタープラン（v1.0）

## 5つの原則
1. Instagram単一アカウントで信頼を積み上げ、DM→商談→成約のファネルを作る
2. 「中小企業・個人事業主のAI導入担当」として専門性を一点集中
3. 人間ループは3点だけ：①企画レビュー ②投稿実行 ③結果記録
4. GitHub がプロジェクトの中枢
5. 30日で仮説検証 → 勝ちパターンに寄せる

## 体制
- オーナー — 戦略・コンテンツ・最終承認
- COOエージェント: `coo`（root `CLAUDE.md`）
- 専門エージェント: 6体（最小スタート）→ 伸びた施策から順次拡張

## アカウント設計

| 項目 | 内容 |
|---|---|
| プラットフォーム | Instagram |
| ターゲット | 中小企業経営者・個人事業主（30〜50代） |
| ポジション | 現場に入れるAI導入の専門家 |
| ゴール | DM問い合わせ → 個別相談 → サービス・講座成約 |

## 収益構造

| 商品 | 価格帯 | フェーズ |
|---|---|---|
| 個別AI導入相談（初回） | 無料 or ¥5,000 | Day1〜 |
| AI業務カスタマイズ支援 | ¥50,000〜 | Day15〜 |
| AI活用講座（グループ） | ¥30,000〜 | Day30〜 |

## 30日後の判定

| 指標 | 続行 | 見直し |
|---|---|---|
| フォロワー | 500名+ | 200名未満 |
| 保存率 | 5%+ | 2%未満 |
| DM問い合わせ | 月5件+ | 0件 |
| 成約 | 月1件+ | 0件 |

→ 3項目以上が見直し基準 → ターゲット・訴求軸を全面再設計。
